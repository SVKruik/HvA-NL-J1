#

To give the platform its unique character, we user custom terminology for things like users. Below are the translations to common vocabulary.

Convention is to always write these terms with a capital letter. 

---

#### Entities

- `admin` = Super user, platform staff
- `keeper` = Burrow-scoped moderator
- `hopper` = Registered user
- `buddy` = Friend
- `tail` = Follower
- `praise` = Vote
- `carrot` = Valuta
- `burrow` = Community
- `reputation` = Karma
- `stray` = Unregistered user/visitor

#### Components

- `tag` = A subject, Keepers can choose these from a predefined list. Used for global filtering.
- `label` = A Burrow-scoped label that can be used to mark posts. Used for scoped filtering.
- `flair` = A Burrow-scoped flair that Hoppers can apply.  Used only cosmetically.
