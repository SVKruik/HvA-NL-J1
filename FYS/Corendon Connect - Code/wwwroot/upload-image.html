• Uploaden

    • Upload afbeeldingen naar de database.
    • Selecteer het type en Profiel ID, en klik de upload knop.
    • Accepteert een maximale grootte van 3 MB.

    • Verwijderen

    • Verwijder records uit de database en/of afbeeldingen uit de uploads map (FTP).
    • Is gelinkt aan het opgegeven Profiel ID boven, en verwijdert dus specifiek voor het profiel in kwestie.
    • Bij het verwijderen van FTP worden alle types afbeeldingen verwijderd.

    • Houd de console in de gaten voor meer informatie.
